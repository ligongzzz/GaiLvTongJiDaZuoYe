解题思路与代码

这道题，同样是用 Python 编写代码求解的。首先使用 numpy 生成 n×1000n\times 1000n×1000 的符合混合高斯分布的随机数矩阵，之后计算出矩阵的均值与方差，并通过循环计算出 UiU_iUi​ ，最后使用 matplotlib 绘制频率分布直方图。
代码如下：

import numpy as np
import matplotlib.pyplot as plt
# Parameters
m1 = -500
s1 = 1
m2 = 100
s2 = 1
p = 0.5
total_size = 1000
data_size = int(input('n:'))
# Data Generator
z = np.zeros((total_size, data_size))
u = np.zeros(total_size)
x = np.random.normal(m1, s1, (total_size, data_size))
y = np.random.normal(m2, s2, (total_size, data_size))
a = np.random.rand(total_size, data_size) < p
z: np.ndarray = x + a * y
# Calculate EZ and DZ
EZ = z.mean()
DZ = z.var()
# Calculate U_i with a loop
for i in range(total_size):
    z_sum = z[i].sum()
    u[i] = (z_sum - data_size * EZ) / np.sqrt(data_size * DZ)
# Show the plot
plt.hist(u, bins=100)
plt.title(f'n={data_size}')
plt.show()
